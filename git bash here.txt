	Linux bash shell을 사용해 볼 수 있는 커맨드 창이다.

▣  $ : prompt라고 하며, 명령어를 입력하기를 기다리는 표식
	여기에 명령어를 입력하고 enter를 입력하면 해당하는 명령을 실행해서 결과를 알려준다.

▣  list : list를 보여 달라는 가장 많이 사용되는 명령어
▣  ls -ad : list 를 보여주되 여러가지 정보를 자세히 보여 달라는 명령어
▣  cd 하위폴더명 : 헌재의 디렉토리(폴더)에서 하위에 있는 디렉토리(폴더)로 이동하라는 명령어   

▣  디렉토리(폴더)에서 . 와 .. 
		디렉토리(폴더)에서 . 와 .. 는 매우 중요한 개념이다
▣  . : 나 자신 
▣  .. : parent(상위)

▣  cd . : 그 폴더 그대로 있으라는 명령어
▣  cd .. : 상위 디렉토리(폴더)로 이동하라는 명령어

▣  cd ../(이동하려는 디렉토리명(폴더명)) : 현재 디렉토리(폴더)와 레벨이 같은 디렉토리(폴더)로 이동하고 싶을 떄 사용하는 이동 명령어 
▣  cd ../../(이동하려는 디렉토리면(폴더명)) : 현재 디렉토리(폴더)보다 상위 레벨의 다른 디렉토리(폴더)로 이동하고 싶을 떄 사용하는 이동 명령어
▣  Tab(탭키) : 두번 누르면 자동완성 기능있음
